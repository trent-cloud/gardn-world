# DECISIONS — Gardn World

Locked decisions for this project. Each entry is a decision made, the rationale, and the cost of reopening.

## Re-open protocol

A locked decision can be reopened if new information genuinely changes the calculus. The protocol is:

1. Flag explicitly: `RE-OPEN CANDIDATE: <decision>`
2. State the case in one paragraph: what's changed that makes this worth revisiting?
3. State the cost of reopening: migration work, doc churn, time.
4. Recommend: reopen now / hold / defer to next quarter.

If Trent (or Richard) agrees to reopen, the decision moves to a `## Under review` section here. If reopening produces a new decision, the old one moves to `## Archive` with the date superseded.
